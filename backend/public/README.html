Store name: POP U
What we sell: Pop dolls
Additional pages:

  1. "Key Chain Products"
  2. "Autographed Product"
What was hard for me: The fact we use React and express and the way to combine them both
The routes:
  1. "/" : for the home page, the main products
  2. "/signedProducts": for the signed products by famous people, an additional page
  3. "/keyProducts": for the key chain products, an additional page
  4. "/admin": for the admin, where he can see all the activities in the website
  5. "/readme": for the readme page
  6. "/cart": for the cart, where we can add/remove an item
How did I make my store secured? By preventing DDOS attack, using "express-rate-limit" module
Did I use react.js? Yes.
